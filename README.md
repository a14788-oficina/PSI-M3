# 📘 Módulo 3 – Exercícios em Python

Este repositório reúne vários exercícios desenvolvidos em **Python**, no contexto do **Módulo 3 de Programação**.

As atividades foram criadas para reforçar conceitos essenciais de programação, com especial foco em lógica, estruturas condicionais e manipulação de dados.

---

## 📂 Organização do Projeto

Os exercícios estão distribuídos por diferentes ficheiros:

### 🧩 Exercícios Iniciais

* `ex01-Ficha1.py` → Apresentação de dados no ecrã
* `ex01-Ficha2.py` → Cálculo da soma de valores inseridos
* `ex01-Ficha3.py` → Avaliação de nota (maior ou igual a 10)
* `ex01-Ficha4.py` → Simulação de um levantamento bancário
* `ex02-Ficha1.py` → Leitura de dados (nome e idade)
* `ex02-Ficha2.py` → Conversão de Celsius para Fahrenheit
* `ex02-Ficha3.py` → Identificação de número par
* `ex02-Ficha4.py` → Classificação etária
* `ex03-Ficha2.py` → Cálculos matemáticos básicos
* `ex03-Ficha3.py` → Verificação de credenciais de acesso
* `ex03-Ficha4.py` → Determinar o maior entre três valores
* `ex04-Ficha4.py` → Aplicação de descontos
* `ex00-Ficha2.py` → Identificação do tipo de variável
* `exp03-Ficha1.py` → Estimativa de idade

---

## 🚀 Como Utilizar

1. Clonar o repositório:

   git clone https://github.com/a14788/3-modulo.git

2. Aceder à pasta do projeto:

   cd 3-modulo

3. Executar um dos ficheiros:

   python nome_do_ficheiro.py
